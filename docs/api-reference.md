# 📑 Referência da API

## 🧑 Usuários

- GET `/v1/user/:id`
- POST `/v1/user`
- PUT `/v1/user/:id`
- DELETE `/v1/user/:id`
- POST `/v1/user/token` (Auth)

## 📁 Categorias

- GET `/v1/category/search`
- GET `/v1/category/:id`
- POST `/v1/category`
- PUT `/v1/category/:id`
- DELETE `/v1/category/:id`

## 🛒 Produtos

- GET `/v1/product/search`
- GET `/v1/product/:id`
- POST `/v1/product`
- PUT `/v1/product/:id`
- DELETE `/v1/product/:id`
